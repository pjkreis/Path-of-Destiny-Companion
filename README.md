# Path-of-Destiny-Companion

Created a simple html app to read aloud the chapters story texts within the Witcher - Path of Destiny boardgame.
The app is not enough to play the game, it only reads the texts to add to the ambiance of the gaming session.

Highly recommend getting an ElevenLabs API Key, since this allows you to use fantasy-themed voices for text-to-speech instead of the default ones available in your browser.

Only 1 tale has been added. More tales to come once I confirm with the game creators this is fine to share.
